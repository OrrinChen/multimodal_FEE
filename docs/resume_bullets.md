# Resume Bullets

## Primary

Built a multimodal financial due-diligence evidence engine over SEC filings, XBRL facts, earnings transcripts, investor-deck evidence, and local raw-document chunks, benchmarking BM25, dense proxy, hybrid retrieval, graph retrieval, and a validator-augmented full engine on 60+ due-diligence tasks.

## Reliability

Designed claim-level validators for citation support, fiscal-period alignment, entity matching, numeric reconciliation, unsupported claims, contradictions, and red-team failure modes; generated auditable memos with evidence tables, numeric reconciliation rows, and trace manifests.

## Product Artifact

Packaged the system into a local CLI, Streamlit demo, portfolio PDF/Markdown report, case studies, and interview-ready documentation without requiring API keys or cloud services.

## Short Version

Built a validator-augmented financial evidence engine that verifies due-diligence claims across SEC/XBRL, transcripts, tables, and investor-deck chart evidence with auditable memos and retrieval failure analysis.
